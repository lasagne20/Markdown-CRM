# Application Mobile CRM

## Description

Développement d'une application mobile native (iOS et Android) pour permettre aux commerciaux de gérer leurs contacts et opportunités en déplacement.

## Fonctionnalités principales

- Gestion des contacts
- Suivi des opportunités
- Agenda intégré
- Synchronisation cloud
- Mode hors ligne

## Stack technique

- React Native
- Firebase pour le backend
- Redux pour la gestion d'état
- API REST

## Risques identifiés

- Délais serrés
- Complexité de la synchronisation offline
- Compatibilité entre versions iOS/Android
