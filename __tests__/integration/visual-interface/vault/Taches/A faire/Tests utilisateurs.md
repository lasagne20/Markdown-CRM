# Tests utilisateurs

## Description

Recueillir les retours des utilisateurs beta pour l'application mobile CRM.

## Plan de test

1. Sélection des bêta-testeurs (10-15 personnes)
2. Formation des testeurs
3. Phase de test (2 semaines)
4. Collecte des feedbacks
5. Analyse et priorisation des améliorations

## Critères d'évaluation

- Facilité d'utilisation
- Performance de l'application
- Stabilité
- Fonctionnalités manquantes
- Design et ergonomie
