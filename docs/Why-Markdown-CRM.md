## ğŸ¤ Making the Switch

### Migration from Traditional CRM

**Step 1:** Export data from current CRM
**Step 2:** Write simple conversion script
**Step 3:** Import to Markdown CRM
**Step 4:** Validate and customize

**Time:** 2-4 hours for typical migration

**Example script:**
```python
import csv
import yaml

# Read Salesforce export
with open('salesforce_export.csv') as f:
    contacts = csv.DictReader(f)
    
    for contact in contacts:
        # Create Markdown file
        filename = f"vault/Contacts/{contact['Name']}.md"
        
        # Write YAML frontmatter
        frontmatter = yaml.dump({
            'fullName': contact['Name'],
            'email': contact['Email'],
            'company': contact['Company']
        })
        
        with open(filename, 'w') as out:
            out.write(f"---\n{frontmatter}---\n\n# {contact['Name']}\n")
```
